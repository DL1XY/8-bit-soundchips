# MN6221

## General

<img src="MN6221AA.png" width="600" height="360">	

Family of ICs for generating electronic melody and sound

### IC model overview

Select the program with M11, M12, M13 (L is ON)

Control methods:

- A: Output while ALON = L
- B: Start output when ALON = PULL DOWN,
- C: Start output when MI pin changes
